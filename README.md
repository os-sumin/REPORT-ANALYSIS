# 사업화분석 보고서 자동 생성 시스템

## 🚀 빠른 시작

### 로컬 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

### 사용 방법
1. **템플릿 다운로드**: 사이드바에서 엑셀 템플릿 다운로드
2. **파일 준비**:
   - 최종보고서 PDF
   - 과제정보 엑셀 (필수: 과제명, 기업명)
   - 재무정보 엑셀 (필수: 과제명, 기업명, 연도)
3. **업로드**: 3개 파일 모두 업로드
4. **생성**: '보고서 생성' 버튼 클릭
5. **다운로드**: 생성된 DOCX 파일 다운로드

## 📋 필수 항목

### 과제정보.xlsx
- ✅ **필수**: 과제명, 기업명
- 📝 선택: 연구기간, 정부지원금, 주관기관

### 재무정보.xlsx
- ✅ **필수**: 과제명, 기업명, 연도
- 📝 선택: 부채총계, 자산총계, 매출액, 매출원가, 매출총이익, 경상개발비, 경상연구개발비, 영업이익, 당기순이익

## 📊 생성되는 보고서 구조

1. **표지 및 목차**
2. **과제 개요** (엑셀 데이터 기반)
3. **기업 정보** (GPT 분석 예정)
4. **기술 성과** (PDF 파싱 예정)
5. **재무 현황** (엑셀 데이터 기반)
6. **시장 환경 분석** (GPT 분석 예정)
7. **경쟁사 분석** (GPT 분석 예정)
8. **사업화 전략 제안** (GPT 분석 예정)
